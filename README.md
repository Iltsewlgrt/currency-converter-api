# Currency Converter API

RESTful API сервис для конвертации валют. Выступает в качестве прослойки между клиентским приложением и сторонним провайдером курсов валют (ExchangeRate API).

## Особенности проекта

- Получение актуальных курсов валют и списка доступных валют.
- Автоматическая регистрация пользователей (cookie-based).
- Сохранение "базовой валюты" пользователя в базе данных.
- In-memory кэширование запросов для оптимизации работы со сторонним API.
- Интерактивная документация Swagger.

## Стек технологий

- **Node.js** & **Express**
- **TypeScript**
- **Supabase** (PostgreSQL)
- **Axios** (запросы к стороннему API)
- **Swagger / OpenAPI** (документация)
